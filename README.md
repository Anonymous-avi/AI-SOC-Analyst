# AI SOC Analyst

An AI-powered Security Operations Center (SOC) Copilot that analyzes cybersecurity logs using Machine Learning, NLP, RAG, and Large Language Models.

## Features (Planned)

- Upload firewall, Windows, Linux, and network logs
- Automatic log parsing
- Anomaly detection using ML
- Entity extraction using NLP
- MITRE ATT&CK mapping
- AI-powered incident investigation
- Incident report generation
- Interactive dashboard
